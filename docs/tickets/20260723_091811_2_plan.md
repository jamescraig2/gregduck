## Phase 1: Preparation & Configuration

1. **Configuration updates**:
   - Install packages and save to devDependencies:
     `npm install -D prettier eslint-config-prettier eslint-plugin-unused-imports husky lint-staged`
   - Add scripts to `package.json`:
     - `"format": "prettier --write ."`
     - `"format:check": "prettier --check ."`
     - `"prepare": "husky"`
2. **Scaffolding / Boilerplate**:
   - Create `.prettierrc`:
     ```json
     {
       "semi": true,
       "singleQuote": true,
       "tabWidth": 2,
       "trailingComma": "all",
       "printWidth": 100,
       "bracketSpacing": true,
       "arrowParens": "always",
       "endOfLine": "lf"
     }
     ```
   - Create `.prettierignore`:
     ```ignore
     .next/
     out/
     build/
     dist/
     node_modules/
     coverage/
     .vercel/
     next-env.d.ts
     package-lock.json
     .gemini/
     .agents/
     ```
   - Create `.lintstagedrc.json`:
     ```json
     {
       "*.{js,jsx,ts,tsx,mjs,cjs}": ["eslint --fix", "prettier --write"],
       "*.{json,md,yml,yaml,css,scss}": ["prettier --write"]
     }
     ```

## Phase 2: Core Implementation

1. **Husky Hook Setup**:
   - Initialize Husky setup in CWD:
     ```bash
     npx husky init
     ```
   - Write the hook template into `.husky/pre-commit`:
     ```bash
     #!/bin/sh
     npx tsc --noEmit
     npx lint-staged
     ```
   - Make sure hook script is executable:
     ```bash
     chmod +x .husky/pre-commit
     ```

2. **Modern ESLint Flat Config Updates**:
   - File: [`eslint.config.mjs`](file:///Users/jamescraig/gh/gregduck/eslint.config.mjs)
   - Code Changes:
     ```javascript
     import { defineConfig, globalIgnores } from 'eslint/config';
     import nextVitals from 'eslint-config-next/core-web-vitals';
     import nextTs from 'eslint-config-next/typescript';
     import unusedImports from 'eslint-plugin-unused-imports';
     import eslintConfigPrettier from 'eslint-config-prettier';

     const eslintConfig = defineConfig([
       ...nextVitals,
       ...nextTs,
       {
         plugins: {
           'unused-imports': unusedImports,
         },
         rules: {
           '@typescript-eslint/no-unused-vars': 'off',
           'no-unused-vars': 'off',
           'unused-imports/no-unused-imports': 'error',
           'unused-imports/no-unused-vars': [
             'warn',
             {
               vars: 'all',
               varsIgnorePattern: '^_',
               args: 'after-used',
               argsIgnorePattern: '^_',
             },
           ],
         },
       },
       eslintConfigPrettier,
       globalIgnores(['.next/**', 'out/**', 'build/**', 'next-env.d.ts']),
     ]);

     export default eslintConfig;
     ```

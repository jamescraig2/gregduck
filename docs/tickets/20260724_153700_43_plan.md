## Phase 2: Core Implementation

1. **`AnimalMarkerClusterer` Component Implementation**:
   - File: `[components/map/AnimalMarkerClusterer.tsx](file:///Users/jamescraig/gh/gregduck/components/map/AnimalMarkerClusterer.tsx)`
   - Logic:
     - Use `'use client'` directive.
     - Call `const map = useMap()` to retrieve the active Google Map instance from `@vis.gl/react-google-maps`.
     - Initialize `clustererRef = useRef<MarkerClusterer | null>(null)` inside `useEffect([map])`.
     - Maintain `markersMapRef = useRef<Map<string, google.maps.marker.AdvancedMarkerElement>>(new Map())` to sync active markers.
     - Diff incoming `markers` array against `markersMapRef` on props update:
       - For removed markers: remove from map and clusterer, delete from map ref.
       - For new markers: instantiate custom DOM element with glassmorphism styling (`glass-panel`), animal image thumbnail (`<img>`), species pill badge, and click listener dispatching `onSelect?.(marker)`.
       - Add new markers to `clustererRef.current.addMarkers(...)`.
     - Clean up on unmount or map change (`clusterer.clearMarkers()`).

2. **`AnimalTeaserDrawer` Component Implementation**:
   - File: `[components/map/AnimalTeaserDrawer.tsx](file:///Users/jamescraig/gh/gregduck/components/map/AnimalTeaserDrawer.tsx)`
   - Structure & Layout:
     - If `animal` is `null`, return `null`.
     - Render fixed semi-transparent backdrop (`<div className="fixed inset-0 bg-black/40 z-40" onClick={onClose} />`).
     - Render bottom sheet container (`fixed bottom-0 left-0 right-0 z-50 p-4 max-h-[33vh] glass-panel bg-slate-900/90 text-white rounded-t-2xl shadow-2xl transition-transform duration-300 ease-out`).
     - Drag Handle: Top pill bar (`<div className="w-12 h-1.5 bg-slate-600 rounded-full mx-auto mb-3 cursor-grab" />`).
     - Touch Handlers: Attach `onTouchStart` (capture `clientY`) and `onTouchEnd` (calculate `deltaY = currentY - startY`). If `deltaY > 60` (px), invoke `onClose?.()`.
     - Distance Calculation:
       - If `userPosition` is non-null, calculate `distM = calculateDistanceMeters(userPosition.lat, userPosition.lng, animal.lat, animal.lng)`.
       - Format string: `distM < 1000 ? `${Math.round(distM)} m away` : `${(distM / 1000).toFixed(1)} km away``.
       - If `userPosition` is null or undefined, omit distance UI or show empty distance wrapper.
     - Details Section:
       - Image thumbnail (`<img src={animal.imageUrl} alt={animal.name} className="w-16 h-16 rounded-xl object-cover" />`).
       - Name (`animal.name`), species (`animal.species`), last encountered time (`animal.lastEncounteredAt`).
       - Next.js `<Link href={`/animals/${animal.id}`}>View Details</Link>`.
       - Close button (`<button onClick={onClose} className="...">âś•</button>`).

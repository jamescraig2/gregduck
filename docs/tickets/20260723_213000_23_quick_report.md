## Summary

Implemented RESTful GET API endpoint (`GET /api/markers`) in `app/api/markers/route.ts` to serve animal location markers bounded by client viewport coordinates (`sw_lat`, `sw_lng`, `ne_lat`, `ne_lng`) or H3 spatial index cell bounds (`h3_indexes`).

- Integrated Zod schema validation with range checks for coordinates (`sw_lat`, `ne_lat` within `[-90, 90]`, `sw_lng`, `ne_lng` within `[-180, 180]`).
- Handled antimeridian crossing queries (`sw_lng > ne_lng`) with wrapped OR longitude conditions.
- Validated H3 cell indexes using `isValidH3Index` helper from `services/location/h3.ts`.
- Mapped database fields from Drizzle ORM `animals` table to response contract (`id`, `name`, `species`, `lat`, `lng`, `h3Index`, `lastEncounteredAt` ISO string, `imageUrl`).
- Added unit and integration tests in `tests/api/markers.test.ts`.

3. **useDebounce Custom Hook**:
   - File: `lib/useDebounce.ts` (Create)
   - Signature:
     ```typescript
     import { useState, useEffect } from 'react';

     export function useDebounce<T>(value: T, delay: number): T {
       const [debouncedValue, setDebouncedValue] = useState<T>(value);

       useEffect(() => {
         const timer = setTimeout(() => {
           setDebouncedValue(value);
         }, delay);

         return () => {
           clearTimeout(timer);
         };
       }, [value, delay]);

       return debouncedValue;
     }
     ```
   - Step-by-step logic description: Leverages `useState` initialized with `value` and `useEffect` setting a timer on `value` or `delay` change, returning `clearTimeout(timer)` as cleanup function to prevent memory leaks and unmounted updates.

## Phase 3: Integration & Testing

1. **Unit & Integration Test Suite for `MapDashboard`**:
   - File: `[tests/components/MapDashboard.test.tsx](file:///Users/jamescraig/gh/gregduck/tests/components/MapDashboard.test.tsx)`
   - Mock `@vis.gl/react-google-maps` hooks/components (`APIProvider`, `Map`, `useMap`) and `@/components/map/useGeolocation`.
   - Test Cases:
     - `renders without crashing with empty markers`
     - `shows loading state when useGeolocation loading=true`
     - `fetches markers from /api/markers when camera bounds change`
     - `debounces rapid camera changes — only one fetch per 300ms window` (`vi.useFakeTimers()`)
     - `sets markers to empty array when /api/markers returns HTTP error`

2. **Unit Test Suite for `AnimalTeaserDrawer`**:
   - File: `[tests/components/AnimalTeaserDrawer.test.tsx](file:///Users/jamescraig/gh/gregduck/tests/components/AnimalTeaserDrawer.test.tsx)`
   - Test Cases:
     - `renders null when animal is null`
     - `renders animal details when animal prop is provided`
     - `calculates and displays formatted distance when userPosition is provided`
     - `hides distance when userPosition is null`
     - `invokes onClose when backdrop overlay is clicked`
     - `invokes onClose when close button is clicked`
     - `invokes onClose on touch swipe down gesture exceeding 60px`
     - `does not invoke onClose on minor touch swipe (< 60px)`
